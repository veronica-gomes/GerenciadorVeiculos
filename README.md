##Sistema de cadastro e gestão de veículos para Desktop

Objetivo facilitar o controle de entrada de automóveis inicialmente planejado para veículos de CARGA e PASSEIO.
No sistema é permitido adicionar/deletar/consultar os cadastros realizados.
